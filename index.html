What is CSS?

  1. CSS Stands for Cascading Style Sheets.

  2. CSS describes how HTML elements are to be displayed on screen, paper, or in other media.

  3. Css is not neccesary CSS saves a lot of work. It can control the layout of multiple pages all at once.

  4. External Style sheets are stored in CSS files.

CSS can be added to HTML documents in 3 ways:

  • Inline - by uisng the style attribute inside HTML elements.
  • Internal - by using a style element in the head section.
  • External - by using a link element to link to an external CSS file.
  • The most common way to add CSS, is to keep the styles in external CSS files. However, in this tutorial we will use inline and internal styles, because it is easier to demonstrate, and easier for you to try it yourself.

When comparing CSS Class vs ID, the difference is that CSS class applies style to multiple elements. ID, on the other hand, applies a style to one unique element, ID is also special that you can use a special URL to link directly to an element and it's used by Javascript.

In CSS , selectors are used to target a specific element or range of elements in a web page. Once a element has been targeted , a style or set of styles can be applied to the element.

There is a wide range of selectors avalaible. Two of the most commonly used are class and ID. Both are used to target the elements to which a style should be applied.
